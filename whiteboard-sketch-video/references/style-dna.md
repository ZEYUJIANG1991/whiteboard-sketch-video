# 风格 DNA（品牌无关的骨架，任何品牌包都不应违反）

## 一句话
纯白背景、黑色抖动细线、大量留白、极简人物、少量彩色手写批注——像随手画在
白板上的一张解释草图。业内通称 "Notion-style illustration"（源头：Notion 早期
插画师 Roman Muradov ← Saul Steinberg/New Yorker 谱系；白板动画传统来自
RSA Animate）。

## 必须
- 纯白背景：不要米色、纸纹、渐变、阴影、噪点。
- 黑色手绘线稿为主体：细线、轻微抖动，黑色永远是画面主导色。
- 大量留白：主体约占 50%，至少 35% 空白。
- 一张图只讲一个核心动作/结构/隐喻；主角必须执行核心动作。
- 中文手写标注 ≤5 处，每处 2-8 字；分色语义见品牌包 palette.md。
- 彩色平涂填色仅限 1-2 个焦点元素（主角 + 至多一个关键物件），其余全部空心。
- 图内出现的数字/术语一律中文化（EBITDA→利润），标注和口播保持一致。

## 绝对不要
商业插画 / PPT 信息图 / 正式流程图 / 可爱卡通海报 / 儿童插画 / 精致扁平矢量 /
科技感 UI / 复杂背景 / 左上角类型标题（"流程图""架构图"之类）/ 把每个节点都
解释清楚的说明书画法。

## 分镜取材原则
不平均配图。优先"认知锚点"：核心判断、反差对比、前后断点、闭环、分流、
常见坑、角色状态变化。每次从内容里重新发明一个"奇怪但成立"的隐喻
（玻璃罩=试点炼狱、拔乱线穿新线=重新布线、只刷客厅不改厨房=散点用例），
不要复用过往构图。
